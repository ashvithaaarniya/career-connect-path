# Career Connect Path

Career Connect Path is an AI-powered career guidance platform that helps students and early-career professionals make informed career decisions. The application provides personalized career recommendations, identifies skill gaps, analyzes resumes, and offers structured learning roadmaps to improve employability.

The platform aims to bridge the gap between academic learning and industry requirements by delivering data-driven insights and actionable career guidance through an intuitive and responsive user interface.

---

## Key Features

- Personalized career recommendations based on user profile and interests
- Resume analysis with AI-powered feedback
- Skill gap assessment for target job roles
- Customized learning roadmap and upskilling suggestions
- Career progress tracking
- Responsive and modern user interface
- Secure user authentication
- Dashboard for managing career insights

## Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/career-connect-path.git
```

### Navigate to the project

```bash
cd career-connect-path
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
